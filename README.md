# Autonomous Driving Perception ML-DL Toolkit

A hybrid Artificial Intelligence application demonstrating both classical Machine Learning and Deep Learning.

## Features

### Machine Learning Module
- Dataset loading
- Multiple classifiers
- Accuracy & confusion matrix

### Deep Learning Module
- Traffic sign recognition
- Lane detection
- Pedestrian detection

## Objective
Simulate the perception layer of a self-driving car using computer vision.

## Datasets
- GTSRB Traffic Sign Dataset
- TuSimple Lane Dataset
- WiderPerson Pedestrian Dataset

## Tech Stack
Python, OpenCV, TensorFlow, Scikit-learn, Tkinter
